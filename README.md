# 🚀 Adarsh Gupta | Full Stack Engineer Portfolio

A visually striking, high-performance professional portfolio. This project showcases **Full Stack excellence** through advanced UI/UX principles, robust architecture, and a specialized technical ecosystem developed over an 8-week intensive development cycle.

## 🛠️ Technical Ecosystem
* **Frontend**: React.js (Vite), HTML5, CSS3
* **Backend & DB**: Node.js, Express.js, MongoDB, SQL
* **Languages**: Java, JavaScript (ES6+)
* **Animations**: Framer Motion (Staggered entrance & hover micro-interactions)
* **State Management**: React Context API (Theme Engine)

## 🌟 Key Engineering Features
* **Interactive Skill Matrix**: A dynamic "Initialize" toggle that reveals the tech stack with spring-physics animations and neon-glow effects.
* **Unified Theme Engine**: System-wide Dark/Light mode synchronization using the Context API for seamless user experience.
* **High-UI Architecture**: Modular, component-based structure designed for maximum scalability and maintainability.
* **Responsive Fluidity**: Fully optimized for the modern web—from mobile devices to ultra-wide desktop monitors.

## 📂 Development Milestones
* **Weeks 1–2**: UI/UX Architecture planning and core layout establishment.
* **Weeks 3–4**: Advanced Section development, Context API integration, and Framer Motion logic.
* **Weeks 5–6**: Backend integration logic, Theme Engine enhancements, and Project Gallery refinement.
* **Weeks 7–8**: Performance optimization, Cross-browser testing, and Final Documentation for deployment.

## 🚀 How to Run Locally
1.  **Clone the repository**:
    ```bash
    git clone [https://github.com/adarsh729/My-Portfolio.git](https://github.com/adarsh729/My-Portfolio.git)
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Launch development server**:
    ```bash
    npm run dev
    ```
