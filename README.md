# Adarsh | High-UI React Portfolio 🚀

A visually striking, modern portfolio website developed during my 8-week internship. This project showcases frontend excellence through advanced UI/UX principles, smooth animations, and a professional personal brand.

## 🌟 Key Features
* **High-UI Hero Section**: Features responsive typography and clear calls-to-action.
* **Dark/Light Theme Toggle**: Implemented via React Context API for a seamless user experience.
* **Animated Project Cards**: Powered by Framer Motion with staggered entrance and hover micro-interactions.
* **Modular Architecture**: Component-based structure for high scalability and maintainability.
* **Responsive Design**: Optimized for all devices, from mobile to ultra-wide desktops.

## 🛠️ Technology Stack
* **Frontend**: React.js (Vite), HTML5, CSS3.
* **Styling**: Modern CSS with Glassmorphism and CSS Grid.
* **Animations**: Framer Motion.
* **State Management**: React Context API.

## 📂 Project Milestones Achieved
* **Weeks 1–2**: UI planning and base layout established.
* **Weeks 3–4**: Section development, Context API integration, and Framer Motion setup.
* **Weeks 5–6**: Theme enhancements and project showcase refinement.
* **Weeks 7–8**: Performance optimization, responsiveness testing, and final documentation.

## 🚀 How to Run Locally
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev` to start the Vite development server.
